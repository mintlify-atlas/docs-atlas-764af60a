<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running in minutes with a complete training and evaluation workflow
  </p>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install required packages:
      
      ```bash
      git clone https://github.com/RaviTejaMedarametla/classification-of-handwritten-digits1.git
      cd classification-of-handwritten-digits1
      pip install -r requirements.txt
      ```
    </Step>

    <Step title="Train your first model">
      Train a KNN classifier on the scikit-learn digits dataset:
      
      ```bash
      python -m research.train_cli --model knn --dataset digits
      ```
      
      <Note>
        The `--dataset digits` option uses the built-in scikit-learn dataset, which doesn't require network access. For MNIST, use `--dataset mnist`.
      </Note>
    </Step>

    <Step title="Run compression experiments">
      Evaluate model compression with feature pruning and quantization:
      
      ```bash
      python -m research.train_cli --model knn --dataset digits --run-compression --run-quantization
      ```
      
      Results will be saved to the `artifacts/` directory as JSON files and plots.
    </Step>

    <Step title="Benchmark with hardware constraints">
      Test model performance under memory and compute constraints:
      
      ```bash
      python -m research.evaluate_cli --model rf --dataset digits --runs 5 --memory-budget-mb 8 --compute-scale 0.75
      ```
      
      This runs 5 repeated experiments with confidence intervals and simulates an 8MB memory budget.
    </Step>
  </Steps>
</div>
